🎮 How to Use in Your Games

Add these to your game HTML files (like aliengame.html, cargame.html, etc.)

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Game</title>
  <link rel="stylesheet" href="animations.css">
  <style>
    :root {
      --primary: #00d9ff;
      --secondary: #0099ff;
    }
    /* Your game styles here */
  </style>
</head>
<body>
  <h1 class="animate-glow">Game Title</h1>
  <div class="animate-slideInUp">Your game content</div>
</body>
</html>

🎨 CSS Variables You Can Use

These variables are defined in animations.css and all CSS files

--primary: #00d9ff - Bright Cyan (main color)
--secondary: #0099ff - Sky Blue (accent)
--accent: #ff006e - Neon Pink (highlight)
/* Use variables in your CSS */
.my-element {
  color: var(--primary);
  border: 2px solid var(--secondary);
  background: rgba(var(--accent), 0.1);
}

⚡ Performance Tips

  • ✅ Use animations sparingly (1-2 per section)
  • ✅ Animations use GPU acceleration (transform, opacity)
  • ✅ Mobile-friendly animations automatically disabled for users who prefer reduced motion
  • ✅ All animations optimized for smooth 60fps performance
  • ✅ Test on real devices for best results


## 🗂️ Reorganization script
This repo now contains `organize-site.js` to automatically move files into folders and rewrite links: 

```bash
node organize-site.js
```

After running it, source files move to:
- `pages/` for HTML
- `assets/styles/` for CSS
- `assets/scripts/` for JS
- `assets/images/` and `assets/media/` for assets

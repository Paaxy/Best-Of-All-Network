🔧 STEP 6: CUSTOMIZE (Optional - 15 minutes)
───────────────────────────────────────────────────────────────

To change colors:

☐ Edit animations.css or any CSS file:
  
  Change this:
  --primary: #00d9ff;  (Current cyan)
  
  To your color:
  --primary: #ff0000;  (Example: red)

To change animation speed:

☐ Edit animations.css:
  
  Change this:
  animation: float 4s ease-in-out infinite;
  
  To faster/slower:
  animation: float 2s ease-in-out infinite;  (2x faster)

🎮 STEP 7: ADD TO YOUR GAMES (15 minutes)
───────────────────────────────────────────────────────────────

☐ For each game file (aliengame.html, cargame.html, etc.):

  1. Add this to <head>:
     <link rel="stylesheet" href="animations.css">
  
  2. Add this to <style>:
     :root {
         --primary: #00d9ff;
         --secondary: #0099ff;
     }
  
  3. Add animations to HTML:
     <h1 class="animate-glow">Game Title</h1>
     <div class="animate-slideInUp">Content</div>

  👉 See COPY-PASTE-TEMPLATES.html for full examples!

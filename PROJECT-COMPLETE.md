### Color Palette
```
Primary:   #00d9ff (Bright Cyan) - Main accent
Secondary: #0099ff (Sky Blue)    - Secondary accent  
Accent:    #ff006e (Neon Pink)   - Highlights
BG Dark:   #0a0e27 (Deep Space)  - Main background
BG Dark:   #050a1a (Void)        - Darker background
Text:      #e0e7ff (Off-white)   - Body text
Muted:     #a0aec0 (Gray-blue)   - Muted text
```

### Key Features
- Glass effect: `rgba(255,255,255,0.08)` + `backdrop-filter: blur(10px)`
- Glowing text: `text-shadow` with animation
- Smooth transitions: `0.3s cubic-bezier(.23, 1, .320, 1)`
- Responsive: `clamp()` for fluid sizing
- Accessible: `prefers-reduced-motion` media query included

## 🔧 Customization Examples

### Change Primary Color
```css
:root {
    --primary: #00d9ff;  /* Your color here */
}
```

### Speed Up Animations
```css
@keyframes float {
    animation: float 2s ease-in-out infinite;  /* Was 4s */
}
```

### Add to Your Games
```html
<link rel="stylesheet" href="animations.css">
<h1 class="animate-glow">Game Title</h1>
<div class="animate-slideInUp">Content</div>
```
